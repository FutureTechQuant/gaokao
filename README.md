# gaokao

面向高考与升学决策的信息聚合项目。

## 项目目标

本项目不仅抓取报考公告，还会逐步覆盖以下主题：

- 报考政策
- 招生信息
- 专业趋势
- 就业质量
- 保研信息
- 学校预算
- 国企要求
- 选调生政策
- 国家重点实验室
- 产业链分析
- 消失专业
- 专业可报考公务员岗位比例

## 目录说明

- `crawler/`：采集、发现、分类、分析、渲染代码
- `config/`：分类规则、种子网址、可信域名
- `data/`：结构化产物
- `docs/`：页面静态产物
- `.github/workflows/`：自动化工作流

## 当前状态

当前仓库为第一版骨架结构，后续逐步填充各专题能力。
